# connected-galaxy-embedded
Embedded part of the "Connected Galaxy" project on RaspberryPi with Docker.

## Start project with: 
$ npm install
$ sudo node index.js

## Embedded
Hardware: RaspberryPi, Wii-Nunchuck.

Software:
Docker Container,
Raspian,
Node.js, Johnny-Five and
Meteor DDP.
